Portfolio Tracker

Indian mutual fund portfolio tracking — live NAV, gains, and encrypted Firebase persistence. The app is not wired yet: below are three UI directions with sample data, for review.

Option A — Dashboard cards →

Summary stat tiles up top, one card per portfolio, allocation bar. Light, airy overview-first layout.

Option B — Dense ledger →

Everything on one page: a single grouped table of all funds across portfolios. Compact, data-first.

Option C — Sidebar app →

Dark app-style UI: portfolios in a left sidebar, detail pane for the selected portfolio.

All three pages are static HTML with hard-coded sample data — no JavaScript, no backend. Buttons and links inside them are decorative. Pick a direction (or a mix) and wiring proceeds from there.
