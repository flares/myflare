# todo

Idea intake and planning space for myflare. Every new idea starts here as a note,
gets fleshed out during a **planning phase**, and — once it's ready to become real —
gets **promoted** to its own top-level sub-project folder (like `padyaalu/` or
`forest-friends/`) and dropped from this list.

This folder exists so planning-phase scratch files (notes, mockups, spikes) don't
clutter the repo root while an idea is still being shaped.

## Workflow

1. **Idea** — logged in `index.html` with a one-line description. No files yet.
2. **Planning** — a note file exists under `notes/<slug>.md` capturing scope,
   open questions, and next steps. Any exploratory mockups/spikes for the idea
   also live under `todo/`, e.g. `todo/<slug>/`.
3. **Promoted** — the idea graduates into its own `<slug>/` folder at the repo
   root, gets added to the root `README.md` sub-projects table and to the home
   page (`index.html`), and its entry here is removed.

## Status board

See [`index.html`](index.html) for the live status of every idea, or open this
folder locally / via GitHub Pages.

## Notes

Planning notes for each idea live in [`notes/`](notes/), one file per idea.
