# Lalitha Sahasranamam breakdown and image generation

- **Status:** Idea
- **Logged:** 2026-07-23

## Summary

Break down the 1000 names (nāmāvali) of Lalitha Sahasranamam individually —
meaning, etymology, context — and generate an accompanying image per name (or
per group of names).

## Open questions

- Source text/commentary to use for meanings (Bhaskararaya's commentary? others?)
- One image per nāma, or per cluster of related nāmas?
- Image generation approach/style — consistent deity iconography across 1000 images?
- Output form: reading app (like `padyaalu/`), a book/PDF, or both?
